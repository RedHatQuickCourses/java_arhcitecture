This page demonstrates the use of the asciidoctor-tabs extension within Antora. Tabs are an excellent way to organize content, whether you need to show code examples in multiple programming languages or separate exercise instructions from their solutions.

Let’s look at how language tabs can be used to compare implementation. For example, if you are displaying how to print "Hello" to the console, you might provide options for both Java and Python. In Java, the process involves defining a class and a main method to output the string. In Python, you can achieve the same result by defining a main function and calling it within an if-name-equals-main block.

Tabs are also very useful for interactive learning, such as separating exercises from their solutions. For instance, you might be given an exercise to write a function that returns the sum of two integers. Once you have attempted the task, you can toggle over to the solution tab to see a reference implementation. In this case, the provided solution uses JavaScript, defining an add function that takes two parameters and returns their sum, followed by a console log statement to verify the result with two and three.
